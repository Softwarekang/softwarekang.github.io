Istio-proxyзӣёе…іжҰӮеҝө

2020-02-04

istio-proxy

Istioд»ЈзҗҶжҳҜеҸҜеңЁе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз«ҜдҪҝз”Ёзҡ„еҫ®жңҚеҠЎд»ЈзҗҶпјҢе№¶еҪўжҲҗеҫ®жңҚеҠЎзҪ‘ж јгҖӮд»ЈзҗҶж”ҜжҢҒеӨ§йҮҸеҠҹиғҪгҖӮ

е®ўжҲ·з«ҜеҠҹиғҪпјҡ
  • еҸ‘зҺ°е’ҢиҙҹиҪҪе№іиЎЎгҖӮд»ЈзҗҶеҸҜд»ҘдҪҝз”ЁеҮ дёӘж ҮеҮҶзҡ„жңҚеҠЎеҸ‘зҺ°е’ҢиҙҹиҪҪе№іиЎЎAPIпјҢд»Ҙжңүж•Ҳең°е°ҶжөҒйҮҸеҲҶй…Қз»ҷжңҚеҠЎгҖӮ
  • еҮӯиҜҒжіЁе…ҘгҖӮд»ЈзҗҶеҸҜд»ҘйҖҡиҝҮиҝһжҺҘйҡ§йҒ“жҲ–зү№е®ҡдәҺеҚҸи®®зҡ„жңәеҲ¶пјҲдҫӢеҰӮHTTPиҜ·жұӮзҡ„JWTд»ӨзүҢпјүжіЁе…Ҙе®ўжҲ·з«Ҝиә«д»ҪгҖӮ
  • иҝһжҺҘз®ЎзҗҶгҖӮд»ЈзҗҶз®ЎзҗҶдёҺжңҚеҠЎзҡ„иҝһжҺҘпјҢеӨ„зҗҶиҝҗиЎҢзҠ¶еҶөжЈҖжҹҘпјҢйҮҚиҜ•пјҢж•…йҡңиҪ¬з§»е’ҢжөҒжҺ§еҲ¶гҖӮ
  • зӣ‘жҺ§е’Ңи®°еҪ•гҖӮд»ЈзҗҶеҸҜд»ҘжҠҘе‘Ҡе®ўжҲ·з«ҜжҢҮж Үе№¶и®°еҪ•еҲ°ж··еҗҲеҷЁгҖӮ
жңҚеҠЎеҷЁз«ҜеҠҹиғҪпјҡ
  • йҖҹзҺҮйҷҗеҲ¶е’ҢжөҒйҮҸжҺ§еҲ¶гҖӮд»ЈзҗҶеҸҜд»ҘйҳІжӯўеҗҺз«Ҝзі»з»ҹиҝҮиҪҪпјҢе№¶жҸҗдҫӣе®ўжҲ·з«Ҝж„ҹзҹҘзҡ„йҖҹзҺҮйҷҗеҲ¶гҖӮ
  • еҚҸи®®зҝ»иҜ‘гҖӮд»ЈзҗҶжҳҜgRPCзҪ‘е…іпјҢжҸҗдҫӣJSON-RESTе’ҢgRPCд№Ӣй—ҙзҡ„иҪ¬жҚўгҖӮ
  • и®ӨиҜҒдёҺжҺҲжқғгҖӮд»ЈзҗҶж”ҜжҢҒеӨҡз§Қиә«д»ҪйӘҢиҜҒжңәеҲ¶пјҢе№¶дё”еҸҜд»ҘдҪҝз”Ёе®ўжҲ·з«Ҝиә«д»ҪйҖҡиҝҮж··еҗҲеҷЁжү§иЎҢжҺҲжқғжЈҖжҹҘгҖӮ
  • зӣ‘жҺ§е’Ңи®°еҪ•гҖӮд»ЈзҗҶеҸҜд»ҘжҠҘе‘ҠжңҚеҠЎеҷЁз«ҜжҢҮж Үе№¶и®°еҪ•еҲ°ж··еҗҲеҷЁгҖӮ

istio proxyе’ҢEnvoyзҡ„е…ізі»

istio proxyиҝҷдёӘйЎ№зӣ®е·ҘзЁӢж—ўеҢ…еҗ«еј•з”ЁдәҶEnvoyзҡ„жәҗз ҒпјҢиҝҳеңЁжӯӨеҹәзЎҖдёҠиҮӘе·ұеҒҡдәҶжү©еұ•пјҢиҝҷдёӘжү©еұ•жҳҜйҖҡиҝҮEnvoy filterпјҲиҝҮж»ӨеҷЁпјүзҡ„еҪўејҸжқҘжҸҗдҫӣпјҢиҝҷж ·зҡ„иҜқе°ұеҸҜд»ҘдҪҝеҫ—proxyд»ЈзҗҶе°Ҷзӯ–з•Ҙжү§иЎҢеҶізӯ–е§”жүҳз»ҷMixerпјҢиҝҷж ·е°ұи§ЈйҮҠдәҶдёәд»Җд№ҲMixerеҸҜд»Ҙиў«и®ҫи®ЎдёәжҸҗдҫӣзӯ–з•Ҙе’ҢйҒҘжөӢзҡ„з»„д»¶пјҢMixer->istio proxy->Envoyиҝҷз§ҚеҪўејҸжқҘжҺ§еҲ¶гҖӮиҝҷж ·йҖҡиҝҮиҝҷдёӘж–№ејҸе°ұиғҪпјҡ

зј–иҜ‘иҝҮзЁӢдҫқиө–ж ‘еҰӮдёӢ

Envoy

istio-proxyжәҗз ҒдёӯжҸҗдҫӣдәҶenvoy.conf.template йҖҡз”Ёй…ҚзҪ®жЁЎжқҝпјҢиҝҷдёӘжЁЎжқҝж–Үд»¶жңҖз»Ҳдјҡз”ҹжҲҗдёҖдёӘenvoyзҡ„й…ҚзҪ®ж–Үд»¶пјҢз„¶еҗҺenvoyеҗҜеҠЁзҡ„ж—¶еҖҷжҢҮе®ҡиҝҗиЎҢгҖӮжЁЎжқҝй…ҚзҪ®ж–Үд»¶дёӯе·Із»Ҹй…ҚзҪ®еҘҪдәҶMixerзӣёе…ізҡ„еҸӮж•°еҰӮmixer_serverпјҢиҝҷдёӘMixerеҜ№дәҺEnvoyжқҘиҜҙе°ұжҳҜдёҖдёӘclusterпјҢеӣ жӯӨжҳҜеңЁcluster_managerйҮҢйқўиҝӣиЎҢз®ЎзҗҶй…ҚзҪ®гҖӮ

  • еҗҺзҪ®жү№йҮҸдёҠжҠҘ

  • жөҒйҮҸд»ЈзҗҶ
  • зӯ–з•ҘжҺ§еҲ¶
  • йҒҘжөӢд»ЈзҗҶ

EnvoyиҜҰи§Ј

InstanceUtil::loadBootstrapConfig(envoy::config::bootstrap::v2::Bootstrap& bootstrap,
                                  Options& options) {
  try {
      // ж №жҚ®й…ҚзҪ®дҝЎжҒҜжҹҘжүҫеҜ№еә”й…ҚзҪ®ж–Үд»¶
    if (!options.configPath().empty()) {
      MessageUtil::loadFromFile(options.configPath(), bootstrap);
    }
    if (!options.configYaml().empty()) {
      envoy::config::bootstrap::v2::Bootstrap bootstrap_override;
      MessageUtil::loadFromYaml(options.configYaml(), bootstrap_override);
      bootstrap.MergeFrom(bootstrap_override);
    }
    MessageUtil::validate(bootstrap);
    return BootstrapVersion::V2;
  } catch (const EnvoyException& e) {
    if (options.v2ConfigOnly()) {
      throw;
    }
    // TODO(htuch): When v1 is deprecated, make this a warning encouraging config upgrade.
    ENVOY_LOG(debug, "Unable to initialize config as v2, will retry as v1: {}", e.what());
  }
  if (!options.configYaml().empty()) {
    throw EnvoyException("V1 config (detected) with --config-yaml is not supported");
  }
  Json::ObjectSharedPtr config_json = Json::Factory::loadFromFile(options.configPath());
  Config::BootstrapJson::translateBootstrap(*config_json, bootstrap);
  MessageUtil::validate(bootstrap);
  return BootstrapVersion::V1;
}
